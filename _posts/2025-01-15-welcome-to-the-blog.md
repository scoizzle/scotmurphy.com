Welcome to my new blog. This is the first post on what I hope will become a running collection of notes,
deep dives, and thoughts on software engineering.

### What to expect

- **Distributed systems** — transactions, consistency models, sagas, and the like
- **.NET deep dives** — the CLR, IL rewriting, AOT, and framework internals
- **Domain-Driven Design** — tactical patterns, aggregates, domain events
- **Tooling** — things I build, things I use, things I wish existed

### How this site works

This blog is built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/)
— completely free. Posts are written in Markdown with YAML front matter. No database, no CMS, no paid hosting.

Check out the [About page](/about/) for more, or browse the [GitHub repo](https://github.com/scoizzle/scotmurphy.com)
if you're curious about the setup.

### Colophon

- Static site generator: Jekyll (stock GitHub Pages version)
- Hosting: GitHub Pages (free tier)
- Domain: scotmurphy.com
- CSS framework: Bootstrap 4
- Icons: Font Awesome 5
